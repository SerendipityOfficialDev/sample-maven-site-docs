Quick reference

Comandi MAVEN per creazione sito per il progetto di DEMO

Task Comando Output root
Creazione documentazione in locale mvn clean site ${project.basedir}/target/site
Pubblicazione su repository GITHUB mvn clean site-deploy GITHUB gh-pages
Pubblicazione su repository AzureDevops mvn clean site-deploy -Psite-azuredevops -Pjavadoc-markdown AZUREDEVOPS Wiki
